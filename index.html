Apa itu Angry Eggs?

Angry Eggs merupakan game sederhana berbasiskan game "BREAKOUT", game klasik yang terkenal dari dulu hingga sekarang. Game ini sangat simple, namun asyik untuk dimainkan dan sering membuat ketagihan. Dengan menampilkan beberapa tokoh karakter dari game fenomenal Rovio, Angry Birds, Breakoutpun bisa disulap menjadi game baru yang lebih menarik dari sebelumnya.

Breakout Game

Breakout adalah sebuah game arcade yang dikembangkan oleh Atari, Inc yang diperkenalkan pada tanggal 13 Mei 1976. Game ini dikonsep oleh Nolan Bushnell dan Steve Bristow, dan masih sering digunakan konsepnya hingga sekarang.
